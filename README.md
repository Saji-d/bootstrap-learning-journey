# Bootstrap 5 Learning Journey

This repository contains all projects I built while completing the course: **The Ultimate Bootstrap 5 Course From Beginners to Advanced**

These projects were built during my **BSS Internship learning journey**, where I practiced building responsive UI components and real-world web applications using **Bootstrap 5** and deployed them using **Vercel**.

This repository includes **all sections and projects from the course**, ranging from fundamentals to full real-world applications.

Concepts covered include:

- Bootstrap 5 fundamentals
- Grid system & layout design
- Responsive design
- Forms & validation
- UI components (Navbar, Cards, Modals, etc.)
- Utility classes
- Dashboard layouts
- Multi-page applications
- Real-world project structuring
- Deployment with GitHub and Vercel
